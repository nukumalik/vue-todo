# Vue Todo

Simple todo app build with vue. Feel free to use, edit, fork and ask.

---

## Getting Started

#### Clone the repo

```bash
$ git clone https://github.com/nukumalik/vue-todo.git
$ cd vue-todo
```

#### Install the requiremet packages

NPM

```bash
$ npm install
```

Yarn

```bash
$ yarn install
```

#### Run the app

NPM

```bash
$ npm run serve
```

Yarn

```bash
$ yarn serve
```
